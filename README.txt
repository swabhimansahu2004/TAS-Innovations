Basic Text Editor
The Basic Text Editor is a simple text editor with essential functionality to create, edit, and format text documents. It offers features like opening, saving, and saving as text files, as well as basic text formatting options.

Features

File Menu

Open: Allows you to open an existing text file.
Save: Saves the current content to the last saved file or prompts you to choose a location if not previously saved.
Save As: Prompts you to select a location and filename for saving the current content to a new file.
Exit: Exits the application. If there are unsaved changes, it prompts you to save them.

Edit Menu

Cut: Cuts the selected text to the clipboard.
Copy: Copies the selected text to the clipboard.
Paste: Pastes the content from the clipboard to the cursor position.
Text Size: Allows you to change the font size of the text.
Text Color: Lets you choose a custom text color.
Format Toolbar

Bold: Toggles the selected text's style to bold.
Italic: Toggles the selected text's style to italic.
Font Family

Font Family: Provides a menu option to select a font family for the text.
Usage
Opening a Text File
Click on "File" > "Open" or use the shortcut Ctrl + O to open an existing text file. A file dialog will appear to select the file you want to open.

Saving a Text File
Click on "File" > "Save" or use the shortcut Ctrl + S to save the current content to the last saved file. If it's the first time saving, it will prompt you to choose a location and file name.
To "Save As," click on "File" > "Save As" or use the shortcut Ctrl + Shift + S. You'll be prompted to select a location and file name.

Exiting the Application
Click on "File" > "Exit" or use the shortcut Ctrl + Shift + X to exit the application. If there are unsaved changes, it will prompt you to save them.
Text Editing and Formatting

You can perform basic text editing operations like cutting, copying, and pasting using the "Edit" menu.

To change the font size, select "Edit" > "Text Size" or use the shortcut Ctrl + T. Enter the new font size in the dialog.
To change the text color, select "Edit" > "Text Color" or use the shortcut Ctrl + Shift + T. Choose a new text color using the color dialog.

Use the "Format Toolbar" to apply text styles such as bold and italic.

Changing Font Family
To change the font family, select "Edit" > "Font Family" or use the shortcut Ctrl + F. A font selection dialog will appear, allowing you to choose the font family for your text.

Building and Running
Compile the Java source code using a Java compiler (e.g., javac Basic_Text_Editor.java).

Run the application (e.g., java Basic_Text_Editor).
